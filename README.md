# Outil Supervision Hybride & SOAR 
**Projet de fin d'études - BTS CIEL Option IR**

##  Présentation
Ce projet est un Proof of Concept réalisé pour l'entreprise **Deletec**. 
L'objectif est de fusionner la supervision réseau (**PRTG**) et la sécurité (**Wazuh**) dans une interface unique avec une réponse automatisée (**SOAR**).

##  Stack Technique
* **Hyperviseur :** VMware Workstation Pro
* **Réseau/Sécurité :** pfSense, Wazuh
* **Monitoring :** PRTG Network Monitor
* **Développement :** Python (Flask/Django), PostgreSQL
* **IoT :** Raspberry Pi 4 + Capteur DHT22

##  Planning
Le projet est découpé en 6 sprints, avec une livraison finale prévue pour le 31 mars 2026.
