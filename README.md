**Projet de fin d'études - BTS CIEL Option IR**

## 📝 Présentation
[cite_start] Ce projet est un Proof of Concept réalisé pour l'entreprise **Deletec** [cite: 31]. 
[cite_start] L'objectif est de fusionner la supervision réseau (**PRTG**) et la sécurité (**Wazuh**) dans une interface unique avec une réponse automatisée (**SOAR**) [cite: 33, 35].

* [cite_start] **Hyperviseur :** VMware Workstation Pro [cite: 103]
* [cite_start] **Réseau/Sécurité :** pfSense, Wazuh [cite: 108, 112]
* [cite_start] **Monitoring :** PRTG Network Monitor [cite: 111]

* [cite_start] **IoT :** Raspberry Pi 4 + Capteur DHT22 [cite: 96, 97]
