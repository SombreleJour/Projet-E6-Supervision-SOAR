# Planning Prévisionnel Projet E6 - Milo MAZZONE
## Outil Supervision Hybride (Monitoring PRTG / SIEM Wazuh) et SOAR

**Période :** 23 Février 2026 - 31 Mars 2026
**Charge totale :** 150 heures

### Semaine 1 : Infrastructure & Réseau (23/02 - 01/03)
- **Lundi 23/02 (4h) :** Installation pfSense et segmentation VLANs (LAN/DMZ).
- **Mardi 24/02 (5h) :** Déploiement des VMs (Windows Server, Win10, Ubuntu).
- **Mercredi 25/02 (4h) :** Configuration routage et règles Firewall pfSense.
- **Jeudi 26/02 (5h) :** Installation VM Gestion (Debian) pour DB et App Web.
- **Vendredi 27/02 (4h) :** Tests de connectivité et sécurisation SSH/RDP.
- **Weekend (4h) :** Schémas réseau (Draw.io) et documentation infra.

### Semaine 2 : Supervision & SIEM (02/03 - 08/03)
- **Lundi 02/03 (5h) :** Installation PRTG Core Server et configuration SNMP/WMI.
- **Mardi 03/03 (5h) :** Déploiement agents supervision et seuils d'alerte.
- **Mercredi 04/03 (6h) :** Installation Stack Wazuh (Manager/Indexer/Dashboard).
- **Jeudi 05/03 (5h) :** Déploiement agents Wazuh et vérification remontée logs.
- **Vendredi 06/03 (4h) :** Configuration règles de corrélation personnalisées.
- **Weekend (3h) :** Dashboards PRTG "Santé du Parc".

### Semaine 3 : Backend & API (09/03 - 15/03)
- **Lundi 09/03 (6h) :** Création Base de Données PostgreSQL (3NF).
- **Mardi 10/03 (6h) :** Développement API Flask/Django et Auth.
- **Mercredi 11/03 (6h) :** Scripts récupération API PRTG.
- **Jeudi 12/03 (6h) :** Intégration alertes Wazuh dans l'API.
- **Vendredi 13/03 (4h) :** Tests unitaires (Postman).
- **Weekend (2h) :** Versioning Git et documentation code.

### Semaine 4 : Frontend & IoT (16/03 - 22/03)
- **Lundi 16/03 (6h) :** Développement Dashboard Web (Widgets & Graphiques).
- **Mardi 17/03 (6h) :** Système de ticketing (Assignation, Statuts).
- **Mercredi 18/03 (6h) :** Configuration Raspberry Pi + Capteur DHT22.
- **Jeudi 19/03 (5h) :** Script Python RPi (Envoi données Temp/Hum).
- **Vendredi 20/03 (5h) :** Intégration IoT dans le dashboard.
- **Weekend (2h) :** Setup Reverse Proxy Nginx.

### Semaine 5 : SOAR & Automatisation (23/03 - 29/03)
- **Lundi 23/03 (6h) :** Développement Scripts Active Response (Isolation VM).
- **Mardi 24/03 (6h) :** Orchestration (Alerte -> SOAR -> Ticket).
- **Mercredi 25/03 (6h) :** Test Scénario Malware (Isolation auto).
- **Jeudi 26/03 (5h) :** Test Scénario Brute Force (Blocage IP).
- **Vendredi 27/03 (4h) :** Débogage final et optimisation.

### Semaine 6 : Métrologie & Clôture (30/03 - 31/03)
- **Lundi 30/03 (5h) :** Mesures Oscilloscope/Multimètre et calcul incertitudes.
- **Mardi 31/03 (5h) :** Finalisation documentation bilingue et livraison.
